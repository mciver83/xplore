Xplore
--------

## Commands

```sh
# Install dependencies
$ yarn

# Start the JS packager;
# IMPORTANT: This must be running when using dev mode.
$ yarn start

# Run the project on an iOS simulator
$ react-native run-ios

# Run the project on an Android simulator
$ react-native run-android

# Run specs
$ yarn test

# Run specs continuously (on file change)
$ yarn test:watch

# Run the linter
$ yarn lint

# Run the linter, fix minor errors automatically
$ yarn lint:fix
```
